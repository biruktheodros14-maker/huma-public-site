# HUMA Public Site

This folder is a separate public GitHub Pages site workspace.

It is intentionally kept out of the private `huma-demo` repository with `.gitignore`.

## Publish steps

1. Create a new public GitHub repository.
2. Copy these files into that repository root.
3. Push to GitHub.
4. In the new repository, enable GitHub Pages from the `main` branch root.

## Notes

- This site is static only.
- Do not copy private code, documents, or datasets into the public repository.
- Replace the placeholder contact text in `index.html` before publishing.
